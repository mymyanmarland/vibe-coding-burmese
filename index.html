Vibe Coding ဆိုတာ ဘာလဲ?

AI ကို အသုံးပြုပြီး ကုဒ်တွေကို မြန်မြန်ဆန်ဆန်နဲ့ စမတ်ကျကျ ရေးသားနည်းကို လေ့လာပါ။

စတင်လေ့လာမည်

အပိုင်း ၁ - အစပျိုးခြင်း (Introduction)

Vibe Coding ခရီးစဉ်

ဒီသင်တန်းမှာ GitHub Copilot, Cursor, Windsurf နဲ့ Lovable စတဲ့ AI tools တွေကို အသုံးပြုပြီး software တွေ၊ website တွေကို ဘယ်လိုဖန်တီးမလဲဆိုတာ လေ့လာရမှာ ဖြစ်ပါတယ်။

  • Developers: ကုဒ်ရေးတာ ပိုမြန်လာမယ်။
  • Non-Technical: ကုဒ်မတတ်ဘဲ app တွေ ဆောက်နိုင်မယ်။

အပိုင်း ၂ - Prompt Engineering (အဆင့်မြင့် နည်းစနစ်များ)

Prompt Engineering ဆိုတာ ဘာလဲ?

AI (ChatGPT, Gemini, etc.) ကို သင်လိုချင်တဲ့ အဖြေအတိအကျ ရအောင် ညွှန်ကြားချက် ပေးတဲ့ လုပ်ငန်းစဉ် ဖြစ်ပါတယ်။

Role-Based Prompts (လူပုဂ္ဂိုလ်ပုံစံ သတ်မှတ်ခြင်း)

AI ကို ကျွမ်းကျင်သူတစ်ယောက်အဖြစ် သတ်မှတ်ပြီး ခိုင်းစေခြင်းဖြင့် ပိုမိုကောင်းမွန်သော အဖြေများ ရရှိနိုင်သည်။

  • "Act as a Senior Python Developer..."
  • "You are an expert UX Designer..."

Multi-step Reasoning (အဆင့်ဆင့် စဉ်းစားခိုင်းခြင်း)

ရှုပ်ထွေးသော အလုပ်များကို တစ်ဆင့်ချင်းစီ ခွဲခိုင်းခြင်း (Chain of Thought) ဖြစ်သည်။

  • ပြဿနာကို အရင်ခွဲခြမ်းစိတ်ဖြာခိုင်းပါ။
  • ပြီးမှ ကုဒ်ရေးခိုင်းပါ။

Best Practices (အကောင်းဆုံး လုပ်ဆောင်ချက်များ)

  • Context: လုံလောက်သော နောက်ခံအကြောင်းအရာ ပေးပါ။
  • Constraints: ကန့်သတ်ချက်များ (ဥပမာ - "ကုဒ် ၂၀ ကျောင်းထက် မပိုစေရ") ထည့်သွင်းပါ။
  • Iteration: တစ်ခါတည်းနဲ့ မရရင် ထပ်ဆင့် ပြင်ဆင်ခိုင်းပါ။

အပိုင်း ၃ - ChatGPT Quick Start (အမြန်လေ့လာခြင်း)

ChatGPT ကို စနစ်တကျ အသုံးပြုခြင်း

ChatGPT သည် Vibe Coding အတွက် အဓိကကျသော အဖော်တစ်ဦး ဖြစ်ပါသည်။

Custom Instructions

AI က သင့်ကို ဘယ်လိုဆက်ဆံရမလဲ၊ အဖြေတွေကို ဘယ်လိုပုံစံ ထုတ်ပေးရမလဲဆိုတာ ကြိုတင်သတ်မှတ်ထားခြင်း ဖြစ်သည်။

  • သင့်ရဲ့ ကျွမ်းကျင်မှုအဆင့်ကို ပြောပြထားပါ။
  • အဖြေကို တိုတိုပဲ လိုချင်သလား၊ အသေးစိတ် လိုချင်သလား သတ်မှတ်ပါ။

Memory & Context

ChatGPT သည် ယခင်ပြောဆိုခဲ့သော အကြောင်းအရာများကို မှတ်မိနေပြီး ပိုမိုနားလည်မှုရှိသော အဖြေများ ပေးနိုင်ပါသည်။

  • Project အဟောင်းတွေအကြောင်း ပြန်မေးနိုင်ခြင်း။
  • ကိုယ့်ရဲ့ စိတ်ကြိုက်ပုံစံ (Preferences) များကို မှတ်ထားပေးခြင်း။

အပိုင်း ၄ - ChatGPT Tools & Features (အဆင့်မြင့် လုပ်ဆောင်ချက်များ)

စွမ်းဆောင်ရည်မြင့် Tool များ

ChatGPT ကို စာရိုက်ရုံတင်မကဘဲ အောက်ပါ tool တွေနဲ့ တွဲဖက်အသုံးပြုနိုင်ပါတယ်။

Advanced Data Analysis

Excel file တွေ၊ CSV တွေကို upload တင်ပြီး data တွေကို graph ဆွဲခိုင်းတာ၊ ခွဲခြမ်းစိတ်ဖြာခိုင်းတာတွေ လုပ်နိုင်ပါတယ်။

Image Generation & Vision

DALL-E 3 သုံးပြီး ပုံတွေဖန်တီးနိုင်သလို၊ ကိုယ့်မှာရှိတဲ့ ပုံကိုပြပြီး "ဒါကို website အဖြစ် ကုဒ်ရေးပေးပါ" လို့ ခိုင်းလို့ရပါတယ်။

Custom GPTs & Store

ကိုယ်ပိုင် AI assistant လေးတွေ (ဥပမာ - Coding Tutor, Logo Maker) ကို ဖန်တီးပြီး အခြားသူတွေသုံးဖို့ Store မှာ တင်ထားနိုင်ပါတယ်။

အပိုင်း ၅ - Models, Plans & Privacy (မော်ဒယ်များ၊ ဈေးနှုန်းနှင့် လုံခြုံရေး)

မှန်ကန်သော Model နှင့် Plan ရွေးချယ်ခြင်း

Models (4o vs o1)

  • GPT-4o: မြန်ဆန်တယ်၊ နေ့စဉ်အလုပ်တွေအတွက် အကောင်းဆုံး။
  • OpenAI o1: ပိုပြီးနက်နက်နဲနဲ စဉ်းစားတယ်၊ ရှုပ်ထွေးတဲ့ ကုဒ်တွေနဲ့ သင်္ချာတွေအတွက် သုံးပါတယ်။

Plans & Privacy

  • Free vs Plus: Plus ($20/mo) က ပိုမြန်တဲ့ model တွေနဲ့ tool တွေအကုန် သုံးလို့ရပါတယ်။
  • Data Privacy: Settings > Data Controls ထဲမှာ "Chat History & Training" ကို ပိတ်ထားရင် ကိုယ့်ရဲ့ data တွေကို AI သင်ယူရာမှာ အသုံးမပြုတော့ပါဘူး။

အပိုင်း ၆ - GitHub Copilot: Getting Started (စတင်အသုံးပြုခြင်း)

AI Pair Programmer ကို စတင်အသုံးပြုခြင်း

GitHub Copilot ဟာ VS Code ထဲမှာ အလုပ်လုပ်တဲ့ အစွမ်းထက်ဆုံး Coding Assistant တစ်ခု ဖြစ်ပါတယ်။

Installation & Setup

  1. GitHub အကောင့်နဲ့ login ဝင်ပြီး Copilot subscription ယူပါ။
  2. VS Code Extension Marketplace မှာ "GitHub Copilot" ကို ရှာပြီး Install လုပ်ပါ။

Key Features

  • Autocomplete: ကုဒ်တစ်ကြောင်းရေးရင် နောက်ဘာလာမလဲဆိုတာ AI က ကြိုပြီး အကြံပေးပါတယ်။ (Tab နှိပ်ပြီး လက်ခံနိုင်ပါတယ်)။
  • Copilot Chat: Ctrl+Shift+I (သို့) Sidebar က Chat မှာ AI နဲ့ တိုက်ရိုက်ဆွေးနွေးနိုင်ပါတယ်။
  • Fix This: ကုဒ်မှာ အမှားရှိရင် AI ကို ပြင်ခိုင်းလို့ ရပါတယ်။

အပိုင်း ၇ - Advanced GitHub Copilot (Chat Modes)

Copilot ကို ပိုမိုကျွမ်းကျင်စွာ အသုံးပြုခြင်း

Copilot သည် ကုဒ်ရေးရုံတင်မကဘဲ တစ်ခုလုံးကို စီမံခန့်ခွဲပေးနိုင်သော အဖော်တစ်ဦး ဖြစ်လာပါသည်။

Getting to know Copilot

Autocomplete ထက်မကသော စွမ်းဆောင်ရည်များ - Refactoring, Explanation နှင့် Unit Test ဖန်တီးခြင်းများ။

Chat Modes & Interfaces

  • Chat View: ရှုပ်ထွေးသော မေးခွန်းများအတွက်။
  • Inline Chat (Ctrl+I): ကုဒ်ရေးနေရင်း တိုက်ရိုက် ပြင်ဆင်ရန်။
  • Terminal Chat: Command line ညွှန်ကြားချက်များ မေးမြန်းရန်။

အပိုင်း ၈ - Context & Precision (တိကျသော နောက်ခံအကြောင်းအရာ)

AI ကို အသိဉာဏ် ပေးခြင်း

AI က သင့် project ကို ပိုမိုနားလည်စေရန် Context ပေးခြင်းသည် အရေးကြီးဆုံး ဖြစ်သည်။

@ Mentions

  • @workspace: Project တစ်ခုလုံးကို အခြေခံပြီး ဖြေခိုင်းခြင်း။
  • #file: သတ်မှတ်ထားသော ဖိုင်ကို ညွှန်းခြင်း။
  • #selection: ရွေးထားသော ကုဒ်အပိုင်းအစကိုသာ ပြင်ခိုင်းခြင်း။

အပိုင်း ၉ - Calculator App Project (လက်တွေ့ လုပ်ဆောင်ခြင်း)

Calculator App တစ်ခုကို Copilot ဖြင့် ဆောက်ခြင်း

ဒီအပိုင်းမှာ Copilot ကို သုံးပြီး အသုံးဝင်တဲ့ app တစ်ခုကို အစကနေ အဆုံးထိ ဖန်တီးပုံကို လေ့လာရမှာ ဖြစ်ပါတယ်။

  • UI Design ကို HTML/CSS နဲ့ ဆောက်ခိုင်းပါ။
  • Logic ကို JavaScript နဲ့ ရေးခိုင်းပါ။
  • အမှားတွေကို AI နဲ့ပဲ ပြန်ပြင်ခိုင်းပါ။

အပိုင်း ၁၀ - Indexing & Instructions (စီမံခန့်ခွဲမှု နည်းစနစ်များ)

Project တစ်ခုလုံးကို AI က နားလည်စေရန်

Workspace Indexing

Copilot က သင့် project ထဲမှာ ရှိတဲ့ ဖိုင်တွေအားလုံးကို ဖတ်ပြီး index လုပ်ထားတဲ့အတွက် "ဘယ် function က ဘာလုပ်သလဲ" ဆိုတာကို ကောင်းကောင်းသိပါတယ်။

Custom Instructions

.github/copilot-instructions.md ဖိုင်ကို အသုံးပြုပြီး project တစ်ခုလုံးမှာ လိုက်နာရမယ့် ကုဒ်ရေးနည်း စည်းကမ်းချက်တွေကို သတ်မှတ်နိုင်ပါတယ်။

အပိုင်း ၁၁ - Cursor AI: Getting Started (စတင်အသုံးပြုခြင်း)

AI-Native Code Editor ဆီသို့

Cursor သည် AI ကို အခြေခံပြီး တည်ဆောက်ထားသောကြောင့် VS Code ထက် ပိုမိုမြန်ဆန်၊ ထက်မြက်ပါသည်။

Why Cursor?

  • VS Code ကို အခြေခံထားသဖြင့် အကျွမ်းတဝင်ရှိခြင်း။
  • AI Tab (Next-generation Autocomplete)။
  • Chat နှင့် Composer mode များ ပါဝင်ခြင်း။

Setup & Migration

  1. Cursor.com မှ download ရယူပါ။
  2. VS Code extensions နှင့် settings များကို တစ်ခါတည်း import လုပ်နိုင်ပါသည်။

Vibe Coding ဆိုတာ ဘာလဲ?

AI ကို အသုံးပြုပြီး ကုဒ်တွေကို မြန်မြန်ဆန်ဆန်နဲ့ စမတ်ကျကျ ရေးသားနည်းကို လေ့လာပါ။

စတင်လေ့လာမည်

အပိုင်း ၁ - အစပျိုးခြင်း (Introduction)

Vibe Coding ခရီးစဉ်

ဒီသင်တန်းမှာ GitHub Copilot, Cursor, Windsurf နဲ့ Lovable စတဲ့ AI tools တွေကို အသုံးပြုပြီး software တွေ၊ website တွေကို ဘယ်လိုဖန်တီးမလဲဆိုတာ လေ့လာရမှာ ဖြစ်ပါတယ်။

  • Developers: ကုဒ်ရေးတာ ပိုမြန်လာမယ်။
  • Non-Technical: ကုဒ်မတတ်ဘဲ app တွေ ဆောက်နိုင်မယ်။

အပိုင်း ၂ - Prompt Engineering (အဆင့်မြင့် နည်းစနစ်များ)

Prompt Engineering ဆိုတာ ဘာလဲ?

AI (ChatGPT, Gemini, etc.) ကို သင်လိုချင်တဲ့ အဖြေအတိအကျ ရအောင် ညွှန်ကြားချက် ပေးတဲ့ လုပ်ငန်းစဉ် ဖြစ်ပါတယ်။

Role-Based Prompts (လူပုဂ္ဂိုလ်ပုံစံ သတ်မှတ်ခြင်း)

AI ကို ကျွမ်းကျင်သူတစ်ယောက်အဖြစ် သတ်မှတ်ပြီး ခိုင်းစေခြင်းဖြင့် ပိုမိုကောင်းမွန်သော အဖြေများ ရရှိနိုင်သည်။

  • "Act as a Senior Python Developer..."
  • "You are an expert UX Designer..."

Multi-step Reasoning (အဆင့်ဆင့် စဉ်းစားခိုင်းခြင်း)

ရှုပ်ထွေးသော အလုပ်များကို တစ်ဆင့်ချင်းစီ ခွဲခိုင်းခြင်း (Chain of Thought) ဖြစ်သည်။

  • ပြဿနာကို အရင်ခွဲခြမ်းစိတ်ဖြာခိုင်းပါ။
  • ပြီးမှ ကုဒ်ရေးခိုင်းပါ။

Best Practices (အကောင်းဆုံး လုပ်ဆောင်ချက်များ)

  • Context: လုံလောက်သော နောက်ခံအကြောင်းအရာ ပေးပါ။
  • Constraints: ကန့်သတ်ချက်များ (ဥပမာ - "ကုဒ် ၂၀ ကျောင်းထက် မပိုစေရ") ထည့်သွင်းပါ။
  • Iteration: တစ်ခါတည်းနဲ့ မရရင် ထပ်ဆင့် ပြင်ဆင်ခိုင်းပါ။

အပိုင်း ၃ - ChatGPT Quick Start (အမြန်လေ့လာခြင်း)

ChatGPT ကို စနစ်တကျ အသုံးပြုခြင်း

ChatGPT သည် Vibe Coding အတွက် အဓိကကျသော အဖော်တစ်ဦး ဖြစ်ပါသည်။

Custom Instructions

AI က သင့်ကို ဘယ်လိုဆက်ဆံရမလဲ၊ အဖြေတွေကို ဘယ်လိုပုံစံ ထုတ်ပေးရမလဲဆိုတာ ကြိုတင်သတ်မှတ်ထားခြင်း ဖြစ်သည်။

  • သင့်ရဲ့ ကျွမ်းကျင်မှုအဆင့်ကို ပြောပြထားပါ။
  • အဖြေကို တိုတိုပဲ လိုချင်သလား၊ အသေးစိတ် လိုချင်သလား သတ်မှတ်ပါ။

Memory & Context

ChatGPT သည် ယခင်ပြောဆိုခဲ့သော အကြောင်းအရာများကို မှတ်မိနေပြီး ပိုမိုနားလည်မှုရှိသော အဖြေများ ပေးနိုင်ပါသည်။

  • Project အဟောင်းတွေအကြောင်း ပြန်မေးနိုင်ခြင်း။
  • ကိုယ့်ရဲ့ စိတ်ကြိုက်ပုံစံ (Preferences) များကို မှတ်ထားပေးခြင်း။

အဓိက AI Tools များ

GitHub Copilot

VS Code ထဲမှာ ကုဒ်တွေကို အလိုအလျောက် ဖြည့်ပေးမယ့် AI ပါ။

Cursor AI

AI ပေါ်မှာ အခြေခံထားတဲ့ အစွမ်းထက် Editor တစ်ခု ဖြစ်ပါတယ်။

Windsurf

AI Agent တွေ ပါဝင်တဲ့ အနာဂတ်ရဲ့ Code Editor ဖြစ်ပါတယ်။

Lovable

ကုဒ်မရေးဘဲ app တွေ ဆောက်ပေးနိုင်တဲ့ AI ဖြစ်ပါတယ်။
